# Avatares da Jornada Espiritual

Coloque aqui **5 imagens** (avatares) dos níveis da jornada, **uma para cada nível**, com exatamente estes nomes:

| Arquivo       | Nível | Nome        |
|--------------|-------|-------------|
| `nivel-1.png` | 1     | **Ouvir**   |
| `nivel-2.png` | 2     | **Seguir**  |
| `nivel-3.png` | 3     | **Permanecer** |
| `nivel-4.png` | 4     | **Frutificar**  |
| `nivel-5.png` | 5     | **Multiplicar** |

O app usa essas imagens no card **Seu nível na jornada** (tela Jornada e perfil), no mesmo estilo dos avatares das disciplinas espirituais (ovelhas).

Formato recomendado: **PNG**, tamanho sugerido ~200x200 px ou maior (quadrado).
